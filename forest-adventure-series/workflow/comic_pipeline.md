
# Forest Adventure Series – Comic Production Workflow

1. Write a script for the episode.
2. Generate each panel individually using the prompt template.
3. Maintain character continuity using the character bible.
4. Save panels as:

panel01.png
panel02.png
panel03.png
...

5. Assemble into a grid layout such as:

1 2 3 4
5 6 7 8

6. Export final comic at 4000–6000 px width.

7. Optional: create animated GIF by sequencing panels with 0.8–1.2s timing.
