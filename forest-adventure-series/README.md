
# Forest Adventure Series

A comic and animation project featuring woodland characters learning about Volatco computing.

## Contents

character-bible/
Defines characters and visual style.

characters/
Reference images for Fox and Hedgehog.

assets/
Volatco board reference.

prompts/
Reusable prompts for generating comic panels.

workflow/
Instructions for producing panel comics and animations.

## Production Method

1. Write episode script.
2. Generate each panel individually.
3. Assemble panels into a comic page.
4. Export GIF or static comic.
