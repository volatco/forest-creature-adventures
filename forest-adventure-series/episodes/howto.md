## How to animate the series via gif

`pip install pillow`

### Folder structure for automation

```
episode-001/
│
├── panels/
│   ├── panel-01.jpg
│   ├── panel-02.jpg
│   ├── panel-03.jpg
│   ├── panel-04.jpg
│   ├── panel-05.jpg
│   ├── panel-06.jpg
│   └── panel-07.jpg
│
└── build_episode_gif.py
```

### Run

`python build_episode_gif.py`

### Optimizations

_faster_

`FRAME_DURATION = 800`

_slower_

`FRAME_DURATION = 1500`

_pacing_

`1000 ms`
