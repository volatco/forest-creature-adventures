
# Volatco Woodland Character Bible
*A visual and narrative guide for the Volatco Forest Computing Adventures*

Style foundation:  
Soft watercolor woodland comic illustration, warm natural palette, hand‑drawn outlines, storybook texture, rounded friendly shapes.

Environment baseline:  
Sunny forest clearing, mossy ground, tree trunks, leafy canopy, warm daylight.

---

# Core Characters

## Fox — The Architect

![Fox Character](/forest-adventure-series/references/fox.jpg)

**Role**  
System architect and teacher of Volatco computing.

**Personality**
- confident
- enthusiastic about technology
- enjoys explaining complex ideas simply
- occasionally amused by Hedgehog’s confusion

**Visual Model**
- small orange fox
- cream chest and muzzle
- fluffy tail with white tip
- large ears
- simple black oval eyes
- rounded cartoon proportions

**Typical Poses**
- pointing while explaining
- holding a Volatco board
- drawing diagrams in the dirt
- nodding confidently

**Speech Style**
Clear explanations of technical ideas.

Example:

> "Each node runs its own tiny Forth machine."

---

## Hedgehog — The Curious Learner

![Hedgehog Character](/forest-adventure-series/references/hedgehog.jpg)

**Role**  
Audience surrogate who asks questions.

**Personality**
- curious
- easily confused by engineering concepts
- humorous observations
- self‑deprecating

**Visual Model**
- round fuzzy hedgehog
- small nose
- tiny feet
- simple black eyes
- soft rounded spines

**Typical Poses**
- counting on fingers
- scratching head
- thinking hard
- imagining absurd comparisons

**Speech Style**
Literal interpretations of Fox's explanations.

Example:

> "That sounds like squirrels running a nut warehouse."

# Recurring Object

## The Volatco Computer

![Volatco Board](/forest-adventure-series/references/volatco-board.jpg)

**Symbolism**
Represents distributed asynchronous computing.

**In Comics**
- shown in thought bubbles
- sitting on tree stumps
- used as the computing device the woodland characters interact with

# Story Themes

Common comic themes:

- distributed computing
- asynchronous systems
- Forth machines
- watchdog timers
- low‑power sensing
- debugging workflows

Humor usually comes from:

- Hedgehog misunderstandings
- animal metaphors for computing

Example:

Fox explains nodes → Hedgehog imagines squirrels running a warehouse.

# Style‑Lock Prompt (Reusable)

Use this block in every generation prompt to maintain visual continuity.

```text
Use the Volatco Woodland Character Bible designs:

Fox: small orange fox, cream chest, fluffy tail with white tip.
Hedgehog: round hedgehog with fuzzy spines.

Environment: sunny woodland forest clearing.

Soft watercolor woodland comic illustration style.
Characters must remain visually identical across panels.
```

---

# Panel‑by‑Panel Comic Workflow

1. Generate **one panel at a time**
2. Reuse the **same character prompt**
3. Change only:
   - pose
   - dialogue
   - panel action
4. Assemble panels into a final grid.

Suggested layouts:

```
1 2 3 4
5 6 7 8
```

or

```
1 2
3 4
5 6
7 8
```

Export final page at **4000–6000 px width** for crisp text and print quality.
